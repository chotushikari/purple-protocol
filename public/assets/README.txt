ASSET STRUCTURE:

1. MUSIC:
   Add your soundtrack as: assets/audio/song.mp3
   Format: MP3
   The track is not included for copyright safety.

2. IMAGES (Memory Frames):
   Add 25-30 images in the img/ folder with this naming:
   
   memory1.jpg
   memory2.jpg
   memory3.jpg
   ... up to memory30.jpg
   
   Recommended specs:
   - Aspect ratio: 16:10 or 3:2 (landscape)
   - Size: 800×500px to 1200×750px
   - Format: .jpg, .png, or .webp
   
   If an image is missing, a placeholder will auto-generate.
   The universe will distribute all images in a 3D node network.
